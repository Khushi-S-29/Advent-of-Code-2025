# Advent of Code 2025

My solutions for Advent of Code 2025.  

## About

[Advent of Code](https://adventofcode.com/2025) is a yearly programming challenge created by Eric Wastl.  

Each puzzle has:
- **Part 1** – the main problem  
- **Part 2** – an extension unlocked after solving part 1  
- **Personal inputs** – each participant receives unique input data

## Repository Structure

```
Advent-of-Code-2025/
├── day1/
│   ├── part1.py      # Solution for Day 1, Part 1
│   ├── part2.py      # Solution for Day 1, Part 2
│   └── input.txt     # Personal puzzle input (Inputs are not tracked in git)
├── day2/
│   ├── part1.py
│   ├── part2.py
│   └── input.txt
└── ...
```
